== Description ==

A Suite of settings to control your BuddyPress site.

== Installation ==

1. Upload plugin folder '/buddysuite/' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to your profile/settings/privacy and check personal settings.
4. Admin can set administer settings under the BuddyPress menu in the Wordpress admin.
